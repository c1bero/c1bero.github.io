Hi, I'm a former mathematician turned vulnerability researcher, mainly interested in Android/Linux kernel exploitation.

During my PhD in Mathematics at [FAU](https://www.fau.eu/), I joined the Capture-The-Flag team [FAUST](https://faust.cs.fau.de/) and dove into the
world of IT security. After my PhD, I first worked as an embedded software developer for network interfaces at [Vector
Informatik](https://www.vector.com), then joined [Secfault Security](https://secfault-security.com/) as a Security Consultant. Now I'm working at
[Exodus Intelligence](https://exodusintel.com/) as a vulnerability researcher.

Having learned so much from the community, I created this website to write some blog posts and give something back. I've also included links to
Secfault Security blog posts I contributed to. Enjoy!
